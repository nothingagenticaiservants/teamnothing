# 🧠 Campus Mind AI

AI-powered multi-agent academic assistant using CrewAI + Google Gemini.

## Agents
Attendance Advisor | Performance Analyst | Exam Coach | Doubt Solver | Placement Mentor | Smart Notes

## Tech Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** FastAPI + CrewAI + Gemini 2.5 Flash

## Run Locally

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
